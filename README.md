# 模块功能细化

## 1. 聊天模块
### 功能点
1. **消息类型**：
   - 支持文本、图片、视频、文件、红包。
   - **无痕聊天**：自动销毁聊天记录（客户端存储临时数据，服务端不持久化）。
2. **机器人聊天**：
   - **免费版**：自动回复、简单规则定制。
   - **收费版**：智能分析、自定义 API 集成。
3. **实时通讯**：
   - 使用 WebSocket 实现消息实时推送。
4. **群组管理**：
   - 支持创建群组、邀请好友、移除成员。
5. **音视频功能**：
   - 支持视频聊天、语音聊天、投票聊天。
6. **存储**：
   - **本地存储**：SQLite 保存聊天记录。支持记录导入导出到其他设备。
   - **云端存储**：收费启用 MinIO 分布式存储。

### 技术栈
- 前端：Flutter 使用 WebSocket 库。
- 后端：Go + Kafka + MinIO。

## 4. 小程序模块
### 功能点
1. 区分用户类型：
   - **个人用户**：支持自定义小程序。

### 技术栈

- 后端：Go + MySQL。

## 5. 购物模块
### 功能点
1. 商品体验：
   - 支持商品展示、多条件搜索。
   - 可通过照片搜索商品。
2. 购物流程：
   - 包括购物车、订单管理、退货流程。
3. 支付接口：
   - 支持 Stripe、PayPal、支付宝、微信支付等国内外支付方式。

### 技术栈
- 前端：Flutter 使用 ListView 和 GridView。
- 后端：Go + MySQL。

## 7. 动画和 UI 优化
### 功能点
1. 页面切换动画：
   - 使用 Flutter 的 PageRouteBuilder 和 Hero 动画。
2. 主题切换：
   - 动态/静态主题切换。

## 8. 支付功能集成
### 功能点
1. 支持多种支付方式：
   - **虚拟货币支付**：支持比特币、以太坊支付（Web3.dart 实现）。
   - **国际支付**：支持 Stripe 和 PayPal。
   - **国内支付**：支持银联、云闪付、支付宝、微信支付。
2. 数字钱包：
   - 管理支付方式、余额、支付记录。
3. 红包功能：
   - 类似微信红包，支持金额拆分并分发。

---

## 9. 数据安全和性能优化
### 功能点
1. 聊天记录：
   - 默认本地存储，收费启用云端存储。
2. 云存储架构：
   - 使用 MinIO 实现分布式存储。
3. 链上交易：
   - 支持硬件钱包或多重签名。
4. 扩展性：
   - 所有模块共享统一后端架构，支持高并发。

## 11. 注册登录模块
### 功能点
1. 登录方式：
   - 手机号接收验证码一键登录。
   - 支持 QR 扫码登录、邮箱、账号登录。
2. 注册：
   - 注册需手机号验证码。
   - 邮箱为可选项（未绑定邮箱将提示风险）。
3. 绑定邮箱：
   - 登录后检测未绑定邮箱时，提示用户绑定。
